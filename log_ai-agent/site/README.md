# CyberLog - AI-powered система мониторинга инцидентов

Современный веб-интерфейс с тёмной минималистичной темой для мониторинга инцидентов безопасности.

## 🚀 Быстрый старт

### Требования
- Node.js 16+ и npm

### Установка и запуск

1. **Установите зависимости:**
   ```bash
   npm install
   ```

2. **Запустите dev-сервер:**
   ```bash
   npm run dev
   ```

3. **Откройте в браузере:**
   ```
   http://localhost:3000
   ```

### Демо-доступ
Для входа используйте любые учетные данные (например: `admin` / `admin`)

## 📦 Доступные команды

```bash
npm run dev      # Запуск в режиме разработки
npm run build    # Сборка для production
npm run preview  # Просмотр production-сборки
```

## 🎨 Возможности

- ✅ **Авторизация** - вход с JWT токенами
- 📊 **Панель управления** - обзор метрик и недавних инцидентов
- 🚨 **Инциденты** - список с фильтрацией и поиском
- 🔍 **Детали инцидента** - информация с AI анализом
- 📈 **Статистика** - аналитика и тренды
- 💬 **Чат с AI** - вопросы и рекомендации
- 📅 **История** - архив с фильтрацией по датам
- 🔔 **Уведомления** - real-time через WebSocket

## 🛠️ Технологии

- **Vue 3** + Composition API
- **Vite** - быстрая сборка
- **Tailwind CSS** - тёмная тема
- **Pinia** - управление состоянием
- **Vue Router** - маршрутизация
- **Axios** - HTTP клиент

## 📱 Страницы

- `/login` - Авторизация
- `/` - Главная панель управления
- `/incidents` - Список инцидентов
- `/incidents/:id` - Детали инцидента
- `/statistics` - Статистика
- `/chat` - AI-ассистент
- `/history` - История инцидентов

## ⚙️ Конфигурация

Backend API настроен через прокси в `vite.config.js`:
- По умолчанию: `http://localhost:8000`
- Для изменения: отредактируйте `vite.config.js` → `server.proxy`
